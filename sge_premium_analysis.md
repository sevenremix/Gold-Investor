# 伦敦金 vs 上海金双折线图：价值分析与机构实践

> 本文档记录了关于"是否应在看板中新增伦敦金/上海金双折线图"的讨论结论和最终设计方案。

---

## 一、原始构想

做一张伦敦金、上海金价格的双折线图（以美元盎司记价），横向对比两个市场的黄金定价能力。
*   **伦敦金 (XAU/USD)**：直接取值，无需转换。
*   **上海金 (Au9999)**：需要做一步换算 → `Au9999 (CNY/克) × 31.1035 / USD_CNH = USD/oz`

## 二、价值评估

| 观察维度 | 价值 | 说明 |
|:---|:---|:---|
| **两条线本身的走势** | ⚠️ 低 | 两条线 99% 的时间完全同步涨跌，看一条和看两条没有区别。你不会从中获得比单看 XAU/USD 更多的多空信号。 |
| **两条线之间的价差/溢价率** | ✅ 高 | 这才是核心。当缝隙突然张大（溢价飙升）或反转（出现折价），意味着跨境资金流、央行干预、或本土恐慌等重大异常。**系统 F3 因子已经在做这件事。** |
| **价差的历史分位数** | ✅✅ 极高 | 如果叠加一条**价差的历史均值线**和**±2σ 波动带**，就能快速判断当前溢价处于历史什么百分位——这才是机构级别的决策利器。 |

## 三、行业实践

这种分析框架在黄金研究领域有一个专门的名字：**SGE Premium / Shanghai-London Gold Spread**。

*   **世界黄金协会 (World Gold Council, WGC)**：旗下的 Goldhub 数据平台长期维护一条 "Shanghai Gold Premium" 指标曲线，用来追踪中国市场的实物需求强度。它被视为**中国黄金实物需求的晴雨表**——溢价扩大 = 中国买盘强劲，溢价收窄/折价 = 需求疲软或监管收紧。
*   **各大投行/研报**：高盛、花旗、瑞银等在黄金研报中经常引用 SGE Premium 作为判断中国央行购金节奏、以及人民币汇率压力的前瞻指标。
*   **量化对冲基金**：部分专注商品套利的基金（如 Citadel Commodities）会实时监控这条价差曲线，在价差极端偏离时做跨境统计套利（Statistical Arbitrage）。

## 四、最终设计方案

**不采用**双折线图（视觉信噪比极低），改为增加一张 **"SGE Premium 历史走势图"**：
*   **横轴**：时间（过去 3 个月日线级别数据）
*   **纵轴**：沪伦溢价率 (%)
*   **叠加元素**：30 日移动均线 + ±2σ 布林带
*   **参考线**：系统 Hard Override 阈值线（+4.0% 和 -1.5%）
*   **与 F3 因子联动**：读者可以直观看到当前溢价在历史分布中的位置，理解系统做出"强制 100% IAUM"或"强制 100% 518660"决策的依据。

**数据来源**：
*   国际金价采用 yfinance `GC=F`（COMEX 黄金期货），日线收盘价
*   上海金采用 FRED `DEXCHUS`（USD/CNY 官方汇率）反算
*   由于免费 API 无法获取 Au9999 完整日线历史，改用 `国际金价 × (1 + 当日估算溢价率)` 的近似方式，或者利用 akshare 获取上海金基准价
