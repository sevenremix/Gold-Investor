# 量化引擎因子看板 (Factor Breakdown) 深度解析

本文档汇总了对流式前端看板（Streamlit UI）中 **Factor Breakdown** 模块所有核心参数的数学推演过程及其在实战中的对应金融含义。同时，对 Layer 1 中的关键调参锚点（TIPS_Neutral 与 TIPS_Range）进行了专项说明。

---

## 第一层：Layer 1 — 仓位管理 (Position Sizing)
该层解答的核心问题是：**“面临当前的宏观经济与情绪面，这笔交易该下多大的注？”**

| 因子名称 | 详细解析与实战含义 |
| :--- | :--- |
| **TIPS Factor (F₁)** | **实际利率得分 / 宏观机会成本**。<br>基于美国10年期TIPS收益率计算。如果目前数值处于 1.88% 等较高位，说明无风险收益率高，宏观环境对黄金（无息资产）偏空。系统会给仓位打负分，起到“踩刹车”的作用。 |
| **Momentum (F₅)** | **动量情绪得分 / 盘口抛压**。<br>这是 RSI 与 KDJ 的综合映射。如果是极其负面的数值，说明国际盘口已被恐慌性砸盘，黄金正在经历非理性的超买超卖过程，面临严重的“流血”。 |
| **Sizing Score** | **综合意愿得分** = `(TIPS × W_tips) - (Momentum × W_mom)`。<br>系统对宏观环境与短期情绪的博弈。例如，由于超卖恐慌深不见底，它带来的“抄底诱惑”可能会强行盖过宏观面的利空。正数代表建议追加买入，负数代表建议观望或减仓。 |
| **Multiplier** | **最终仓位系数** = `1.0 + (Sizing Score × 0.5)`。<br>执行指令：以此倍数乘以您的“标准单次仓位金额”。如果是 1.10x，意味着系统认为抓到了黄金坑的抄底机会，建议采取 10% 溢价建仓。 |

### 🛠️ 深度聚焦：F₁ 的核心参数 (`TIPS_Neutral` & `TIPS_Range`)

计算 F₁ 实际需要 3 个参数协同：`clip((TIPS_Neutral - TIPS_Current) / TIPS_Range, -1.0, 1.0)`。

#### 1. TIPS_Neutral: 均衡名义实际利率 (系统默认 = 1.5%)
*   **金融语境**：它是系统的**“价值锚点”**。历史长周期看，美国 10 年期 TIPS 的中枢大致落在 1.0% - 2.0% 之间。
*   **牛熊分界线**：高于 1.5% 系统认为是“利空金价”（持有黄金成本太高）；低于 1.5% 认为是“利好金价”。
*   **调参思路**：如果您认为未来美国会长期处于高通胀高息环境，可调高该值使其适应“新常态”；若认为将重返零利率，则应下调。

#### 2. TIPS_Range: 灵敏度缩放系数 (系统默认 = 2.0%)
*   **金融语境**：它就像是汽车的**“转向比”阻尼器**。它起到类似标准差的作用，用于将无量纲的利率偏离转化为 `[-1, 1]` 的标准化得分。
*   **阻尼器效应**：
    *   设为 2.0% 时，利率每偏离 1.0%，得分变动 0.5。只有偏离整整 2% 才会打满正负 1 满分。
    *   调小（如 1.0%）：系统进入“全油门”，极度敏感，美联储细微的水花都会导致仓位飙升。
    *   调大（如 5.0%）：系统变成重装甲，过滤掉寻常波动，只对史诗级宏观异动产生反应。

## 第二层：Layer 2 — 资产路由 (Vehicle Routing)
该层解答的核心问题是：**“决定下注后，这笔钱该换汇去买美股的 IAUM，还是直接用人民币买本土的 518660？”**

> *注：Layer 2 的所有得分逻辑为：**分数越正，越偏向出海买 IAUM；分数越负，越偏向留守本土买 518660***。

| 因子名称 | 详细解析与实战含义 |
| :--- | :--- |
| **F₂ FX × W₂** | **汇率偏离防线**。<br>衡量当前汇兑风险。如果在人民币强势升值阶段换汇（USD/CNH 大幅跌破 MA200），未来极易吃汇率贬值的暗亏。此时系统会给出巨大的负面扣分，**强行阻拦您的资金出海**。 |
| **F₃ SGE × W₃** | **沪伦溢价纠偏**。<br>对比国内现货黄金（Au9999）与国际金价。如果国内不仅没有常态的夸张溢价反而处于相对贴水，留在本土买金就不吃亏，系统会继续倾向于本土（负分）。 |
| **F₄ Friction × W₄** | **ETF 场内摩擦系数**。<br>通过 518660 盘口现价对比 IOPV 计算折溢价率。如果出现如 -1.0 满格拉跨的极端情况，这属于极其重大的**折价套利信号**（市场非理性低价抛售）。系统会打出极端的负分，要求立刻抄底本土折扣货。 |
| **Routing Score R** | **路由总分** = `F₂ + F₃ + F₄`。<br>计算向离岸美元市场分配资金的百分比：`50 + (R / 2)`。<br>高度负数的 R 值（如 -43分）会把高达百分之七十以上的资金死死锁在国内的 518660 上，从而在赚取黄金收益的同时吃透境内的双重套利缓冲垫。 |

## 结语：冷酷的狙击手逻辑

透过这套全参数的透视图结构，可以看出量化引擎并非单边押注的赌徒，而是一位权衡利弊的冷酷狙击手。在系统给出建议（例如：“**放大 1.1 倍的资金量，并将七成直接抄底国内由于非理性抛售打成深坑的518660，剩下三成去定投美股 IAUM**”）的背后，隐藏的是通过一层一层数学算式的过滤，最终剥离了情绪盲区，捕捉到了汇率与流动性的套利空间。

## 附录：AI 视角下的系统局限性与风险评估

作为本系统的辅助开发者，以下是对当前这套量化引擎最真实、客观的自我评价：

### ❌ 真实存在的局限与风险

**1. 最关键的局限：这套策略的回测为零**
- 整个双层模型的权重（40/40/20、0.6/0.4 等参数）全凭**经验性拍板**，目前**没有经过任何历史数据回测验证**其有效性。这在量化领域是严重的缺陷。一套没有历史回测支撑的评分系统，本质上只是“用数学公式包装的主观直觉”。

**2. 518660 折价率的信号有待商榷**
- 当 Factor F₄（Friction）打出 -1.0 满格分时，系统会建议大力买入高折价的 518660。但需要警惕的是，大幅折价有时恰恰是市场对某种重大风险的预警（如赎回抛压、底层流动性危机）。若简单以此视为“免费的午餐”存在逻辑黑盒风险。

**3. 数据源依赖比较脆弱**
- 本系统高度依赖新浪财经、美联储 FRED、雅虎财经等免费开源接口。这些源站随时可能更改规则或屏蔽请求（尤其是跨境抓取）。具备一定的异常捕获能力，但距离工业生产级别的稳健性仍有差距。
